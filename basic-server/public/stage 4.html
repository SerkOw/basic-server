Stage 4

The final stage: Polymer + D3 + API data

In the final our plan on connecting the working elements to an API had been successful. Through Yahoo's Historical Stock price data API, we were able to connect a D3 visualisation and Polymer element to an API Even though we were not able to accomplish this our initial goal of the sentiment analysis of Twitter messages, we managed to build an element that promises a certain reusability and customisablity and hold the capacity to be extended and used in (digital) humanities research on a social media platform like Twitter.
