Four Dude’s Company Final project: The Process

This will be a short progress documentation progress of the final portfolio “Four Dude’s Company”, consisting of:
Serkan Yildizeli, Leon Meuwese, Jan Zuilhof and Sander van Haren.

For our portfolio we intended to integrate the Twitter Search API in a way that it would connect to a selection menu, which contains a predetermined list of stock market definitions/names of corporations/brand. This was a replacement of the initial idea of letting users type brand names that would then be converting these into stock market name versions and thereby initiating the stock price graph, which would constitute to the second part of the component. However, after a long time exploring the possibilities and several difficulties we encountered, we decided to change plans again.

For our final portfolio we have decided to create a page with several stages that show our development.
